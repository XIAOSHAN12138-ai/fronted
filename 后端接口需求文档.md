## 一、基础信息

| 项目 | 说明 |
|------|------|
| 基础路径 | `/api/v1` |
| 请求格式 | JSON |
| 响应格式 | JSON |

```json
  "data": null
}
```

### 2. 图片生成

**接口地址：** `POST /api/v1/generate/image`

**功能说明：** 根据提示词生成图片

**请求参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| prompt | string | 是 | 提示词描述，如"一只可爱的猫咪" |
| model | string | 是 | 模型ID，如"image_5.0_lite" |
| ratio | string | 是 | 比例：1:1, 16:9, 9:16, 4:3, 3:4, 21:9, 3:2, 2:3 |
| resolution | string | 是 | 分辨率：2K, 4K |
| count | int | 否 | 生成数量，默认3，范围1-5 |
| reference_image | string | 否 | 参考图base64编码（可选） |

**请求示例：**
```json
{
  "prompt": "一只可爱的猫咪，毛茸茸的，坐在草地上",
  "model": "image_5.0_lite",
  "ratio": "1:1",
  "resolution": "2K",
  "count": 3
}
```

**响应示例：**
```json
{
  "code": 200,
  "message": "生成任务已创建",
  "data": {
    "task_id": "task_1234567890",
    "status": "processing"
  }
}
```

**调用时机：** 用户输入提示词，点击发送按钮

### 7. 查询任务状态（轮询）

**接口地址：** `GET /api/v1/generate/task/{task_id}`

**功能说明：** 查询生成任务的执行状态和结果

**路径参数：**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| task_id | string | 是 | 任务ID |

**响应示例 - 进行中：**
```json
{
  "code": 200,
  "message": "查询成功",
  "data": {
    "task_id": "task_1234567890",
    "type": "image",
    "status": "processing",
    "progress": 45,
    "result": {},
    "created_at": "2024-01-15T10:00:00Z",
    "completed_at": null
  }
}
```

**响应示例 - 已完成：**
```json
{
  "code": 200,
  "message": "查询成功",
  "data": {
    "task_id": "task_1234567890",
    "type": "image",
    "status": "completed",
    "progress": 100,
    "result": {
      "images": [
        {
          "id": "img_001",
          "url": "https://cdn.example.com/generated/img1.png",
          "thumbnail": "https://cdn.example.com/generated/img1_thumb.png"
        },
        {
          "id": "img_002",
          "url": "https://cdn.example.com/generated/img2.png",
          "thumbnail": "https://cdn.example.com/generated/img2_thumb.png"
        }
      ]
    },
    "created_at": "2024-01-15T10:00:00Z",
    "completed_at": "2024-01-15T10:00:30Z"
  }
}
```

**响应示例 - 视频任务完成：**
```json
{
  "code": 200,
  "message": "查询成功",
  "data": {
    "task_id": "task_1234567890",
    "type": "video",
    "status": "completed",
    "progress": 100,
    "result": {
      "video": {
        "id": "video_001",
        "url": "https://cdn.example.com/generated/video1.mp4",
        "thumbnail": "https://cdn.example.com/generated/video1_thumb.png",
        "duration": 5
      }
    },
    "created_at": "2024-01-15T10:00:00Z",
    "completed_at": "2024-01-15T10:02:00Z"
  }
}
```

**状态值说明：**
- `pending` - 等待处理
- `processing` - 正在生成
- `completed` - 生成完成
- `failed` - 生成失败

**调用时机：** 创建任务后，前端每2秒轮询一次，直到状态为 completed 或 failed

## 四、技术对接说明

### CORS配置
后端需要配置跨域，允许前端域名访问：
```
Access-Control-Allow-Origin: *
Access-Control-Allow-Methods: GET, POST, PUT, DELETE, OPTIONS
Access-Control-Allow-Headers: Content-Type, Authorization
```

### 异步处理机制
1. 所有生成接口都是异步的，创建任务后立即返回 task_id
2. 前端通过轮询 `GET /api/v1/generate/task/{task_id}` 查询进度
3. 建议轮询间隔：2秒
4. 建议超时时间：60秒

### 文件上传
- 图片/视频建议先压缩再转 base64
- 单个请求体大小建议不超过 10MB
- 大文件建议先上传到对象存储，再传 URL

## 六、接口优先级

| 优先级 | 接口 | 说明 |
|--------|------|------|
| 🔴 P0 | GET /api/v1/models | 必须，页面加载需要 |
| 🔴 P0 | POST /api/v1/generate/image | 必须，核心功能 |
| 🔴 P0 | GET /api/v1/generate/task/{id} | 必须，查询结果 |
| 🟡 P1 | POST /api/v1/generate/video/text-to-video | 重要，视频功能 |
| 🟡 P1 | POST /api/v1/generate/digital-human | 重要，数字人功能 |
| 🟡 P1 | POST /api/v1/generate/voiceover | 重要，配音功能 |
| 🟡 P1 | POST /api/v1/generate/agent | 重要，Agent功能 |
| 🟢 P2 | POST /api/v1/auth/login | 建议，用户系统 |
| 🟢 P2 | GET /api/v1/user/quota | 建议，额度显示 |
| 🔵 P3 | 图生视频、首末帧视频 | 可选，高级功能 |
